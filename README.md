# Prior Authorization Workflow Simulator

Day 26 of the #60DayClaudeChallenge

## Overview

This project simulates the healthcare Prior Authorization process used by providers and insurance companies before approving treatments, procedures, medications, or diagnostic services.

The simulator allows users to move patient cases through different workflow stages while understanding how documentation, payer review, and appeals impact the final decision.

## Features

- Patient scenario selection
- Medical necessity evaluation
- Documentation checklist
- Prior Authorization submission
- Payer review workflow
- Approval, pend, and denial decisions
- Appeals and peer-to-peer review
- Workflow timeline tracking
- Educational healthcare insights
- Interactive dashboard
